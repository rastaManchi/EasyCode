1.
Адаптировать свой сайт-блог для мобильных устройств. Основной задачей является использование медиазапросов CSS для изменения макета сайта на экранах с шириной менее 768 пикселей.
Нужно перестроить сетку, чтобы элементы располагались в одну колонку,
увеличить размеры шрифтов для удобства чтения и убедиться, что все интерактивные элементы, кнопки и ссылки,
остаются достаточно крупными для комфортного нажатия на сенсорных экранах.
2.
Адаптировать свой сайт-блог для планшетов и мобильных устройств.
Необходимо продумать промежуточный макет для планшетов, а также перестроить сетку, чтобы элементы располагались в одну колонку на мобильных устройствах. Также важно проверить, чтобы изображения корректно изменяли размер в зависимости от ширины экрана.

Материал для подготовки:
https://docs.google.com/document/d/1lDDlf63XZCMZJSPbReBHRM9iwC4ylYNQDApYPYn0_wQ/edit?usp=sharing 

3.
Реализовать полностью адаптивный дизайн, который учитывает все виды устройств — от мобильных телефонов до больших настольных мониторов.
Необходимо создать несколько уровней медиазапросов для различных диапазонов ширины экрана, обеспечивая оптимальное отображение контента.
Особое внимание уделяется адаптивным изображениям: попробовать использовать атрибуты srcset и sizes, чтобы загружать изображения подходящего разрешения в зависимости от устройства пользователя. 

Материал для подготовки:
https://docs.google.com/document/d/1lDDlf63XZCMZJSPbReBHRM9iwC4ylYNQDApYPYn0_wQ/edit?usp=sharing
