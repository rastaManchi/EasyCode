Базовый URL

http://ваш-домен/api/v1/

Аутентификация

Для создания, обновления и удаления постов требуется токен.

Authorization: ваш_токен

Эндпоинты

GET Получение постов

/api/v1/posts

Параметры (query):

  • page (int) — номер страницы (по умолч. 1)
  • per_page (int) — постов на странице (по умолч. 10)
  • category_id (int) — фильтр по категории
  • sort_by (string) — поле сортировки: created_at, title
  • order (string) — направление: asc, desc
  • q (string) — поиск по заголовку и содержанию

Пример: /api/v1/posts?category_id=1&sort_by=title&order=asc&q=python

GET Получение одного поста

/api/v1/posts/{id}

POST Вход в систему

/api/v1/login

Тело запроса (JSON):

{
    "email": "user@example.com",
    "password": "password123"
}

POST Создание поста (токен)

/api/v1/posts

Тело запроса (JSON):

{
    "title": "Заголовок",
    "content": "Содержание...",
    "category_id": 1
}

PUT Обновление поста (токен)

/api/v1/posts/{id}

Тело запроса (JSON):

{
    "title": "Новый заголовок",
    "content": "Новое содержание...",
    "category_id": 2
}

DELETE Удаление поста (админ-токен)

/api/v1/posts/{id}

GET Категории

/api/v1/categories
