Создание виртуальной среды: `python -m venv .venv`

Активировать виртуальную среду: `.venv\Scripts\activate`

Если виртальная среда не запустилась -- перезапустить терминал

Установили зависимости: `pip install <Название модуля>`

Модули: `flask, python-dotenv, pytest, pytest-flask`

Создать requirements.txt: `pip freeze > requirements.txt`
