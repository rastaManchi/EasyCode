Вирт среда: `python -m venv .venv`
Установить Django: `pip install Django`
Создать проект: `django-admin startproject BlogSite`

Запуск сайта: `python manage.py runserver`
Создать приложение: `python manage.py startapp main`
Выполнить миграции: `python manage.py migrate`
Создать админа: `python manage.py createsuperuser`
Создать файл для миграции: `python manage.py makemigrations`
